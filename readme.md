Proyecto: E-commerce o carrito de compras

Descripción:

Tejidos Adry es una página web básica desarrollada como parte de un curso de Front-End, inspirada en un emprendimiento de tejidos hechos a mano. La página busca transmitir la estética cálida y artesanal del empredimiento, ofreciendo una navegación entre las diferentes colecciones de productos.
La página está estructurada con HTML y CSS, utilizando las etiquetas <header>, <main>, y <footer> para organizar el contenido.

El sitio cuenta con tres páginas principales
- index.html: página de inicio
- crochet.html: sección colección de prendas tejidas a crochet
- 2agujas.html: sección colección de prendas tejidas a dos agujas

En la página principal agregué un formulario de contacto utilizando **Formspree**. El que cuenta con bordes suaves y efecto hover

== Estilo y diseño ==
- Tipografía: dancing script para títulos y poppins para textos y botones
- Colores principales: gamas de violetas y lavandas
- Cards: con imágenes centradas, sombra y animación
- Banners: con texto centrado y bordes redondeados
- Formulario: con caja blanca con sobra y bordes redondeados

== Multimedia ==
- Imágenes del emprendimiento
- Mapa integrado con Google Maps adaptable
